# Manika

MANIKA adalah UMKM yang dikelola oleh mahasiswa dan bergerak di bidang produksi gelang manik-manik.
Setiap produk dirancang untuk membawa cerita, kenangan, serta ketenangan dalam keseharian.

Produk MANIKA dibuat dalam jumlah terbatas dengan penuh perhatian, ketelitian, dan semangat
kolaborasi yang hangat, sehingga setiap gelang memiliki nilai dan makna tersendiri.

## Tujuan Website
Website ini dikembangkan sebagai **media informasi digital** untuk memperkenalkan UMKM MANIKA,
menampilkan produk, serta mendukung kegiatan pemasaran secara online.

## Teknologi yang Digunakan
- React
- TypeScript
- Vite
- Tailwind CSS
- GitHub Pages
